## 4. STRIDE 위협모델링 대응 준비

| 위협 | 설명 | 현재 대응 |
|------|------|----------|
| Spoofing (신원위조) | 기기/사용자 사칭 | 디바이스 시리얼+비밀번호, Bcrypt 해싱, JWT 인증 |
| Tampering (변조) | 전송 데이터 변조 | MQTTS(TLS 8883), WSS, HTTPS 전 구간 암호화 |
| Repudiation (부인) | 행위 부인 | 감사 로그 (로그인/로그아웃/기기등록/비밀번호변경) |
| Information Disclosure (정보유출) | 민감 데이터 노출 | AES-256-CBC 의료정보 암호화, SSL DB 연결, .env 분리 |
| Denial of Service (서비스거부) | 대량 요청 공격 | Rate Limiter 정의됨 (API 적용 필요) |
| Elevation of Privilege (권한상승) | 비인가 기능 접근 | RBAC 3단계, 미들웨어 토큰 검증, 좀비 세션 감지 |
